Google Offers 100,000$ For Startups

Published by Danushka Herath on December 15, 2018December 15, 2018

Reading Time: 3 minutes

අද ලිපියෙන් කියන්න යන්නේ Startups වලට වාසි ආරංචියක්. මේක දැන් ටිකක් පරණයි. හැබැයි ගොඩක් දෙනෙක් තාම දන්නේ නෑ මෙහෙම Offer එකක් තියෙනවා කියලා. මේ වගේ Program එකක් පටන් ගන්න මූලික හේතුව වෙලා තියෙන්නේ දැනට Startup එකක් ආරම්භ කරත් මූල්‍යමය අපහසුතා නිසා අවශ්‍ය කරන Compute Engines, Machine Learning Kits, Databases වගේ පහසුකම් ලබා ගන්න තියෙන අපහසුතා නිසා නතර වෙලා තියෙනවා නම් ඒ Projects වලට අවශ්‍ය පහසුකම් දෙන්න වගේම අලුත් Startup පටන් ගන්න Motivation එකක් දෙන්න. 

මේ Credits යොදා ගන්න පුළුවන් Main Services කිහිපයක්ම තියෙනවා. 

App Engine / Compute Engine

මේ Service එක භාවිතා කරලා ඔයගෙ දැනට තියෙන Project එකක් Google Cloud වල Host කර ගන්න පුළුවන්. App Engine එකක සහ Compute Engine එකක තියෙන වෙනස තමයි App Engine එකේදි ඔයාගෙ Project එක මොන වර්ගයේ ඒ කියන්නේ මොන Language එකෙන් හදලා තියෙන Project එකක්ද කියලා තෝරගත්තාම ඒකට අදාල Environment එක Google එකෙන්ම හදලා දෙනවා. අපිට Default ඒ Environment එක භාවිතා කරලා අපේ Project එක ලේසියෙන්ම Host කරගන්න පුළුවන්. හැබැයි Compute Engine එකකදි අපිට කැමති OS එකකින් Server Machine එකක් ගන්න පුළුවන්. ඊට පස්සේ අපිට අවශ්‍ය කරන Application, Dependencies Install කරගන්න ඕන. 

 Google Cloud AI

Projects වලට අවශ්‍ය කරන AI Models Train කරගන්න මේ Cloud AI එක Use කරන්න පුළුවන්. Local Machine එකක Model එකක් train කරනවට වඩා වේගයෙන් Google Computers වලින් තමන්ගේ Model එකක් Train කරගන්න මේ මගින් අවස්ථාවක් හම්බෙනවා. ඒ වගේම අපිට දැනට Train කරලා තියෙන Models පාවිච්චි කරන්නත් පුළුවන්. Natural Language Recognition, Image Recognition, Video Analysis, Translation Services වලට මේ Service එක භාවිතා කරන්න පුළුවන්. 

 Google Map Platform

Google Maps Platform එක භාවිතා කරලා අපිට නොදන්න තැනට විස්තර, යම් ස්ථානයකට යන Direction එක. ස්ථාන දෙකක් අතර දුර වගේ ගොඩක් දේවල් අපේ Application එකකට Integrate කර ගන්න පුළුවන්. 

 G Suit

Google එකෙන් දෙන මේ Services දැනටමත් ඔයා යම් Limited Conditions යටතේ ඔයාලා භාවිතා කරනවා ඇති. උදාහරණයක් විදියට Google Drive එකේ Free දෙන 15GB ප්‍රමාණය භාවිතා කරනවා ඇති. Google එකේ Credits භාවිතා කරලා අපිට පුළුවන් මේ Limits තව දුරටත් වැඩි කරගන්න. 

Apply කරන්නේ කොහොමද?

Apply කරන්න පුළුවන් Packages 3ක් තියෙනවා. තමන්ගේ Startup එකේ Stage එක අනුව ඒකට Apply කරන්න පුළුවන්.

  1. Start Package – 3,000$ (1 year)
  2. Spark Package – 20,000$ (1 year)
  3. Surge Package – 100,000$ (1 year)

( මේ Credits භාවිතා කරන්න පුළුවන් Google Cloud එක ඇතුලේ තියෙන Services වලට විතරයි. ඒ කියන්නේ ඔයා Credit Card එක Link කරාට Credits ඔයාගේ Card එකට Deposit වීමක් වෙන්නේ නෑ. )
