අද ලිපියෙන් කතා කරන්න යන්නේ දැනට Windows මෙහෙයුම් පද්ධතියක් එක්ක වැඩ කරන Lap එකක් නැත්නම් Desktop එකකට Ubuntu Install කරගන්නේ කොහොමද කියලා. මේ වැඩේ කරන්න ප්‍රධාන ක්‍රම දෙකක් තියෙනවා. එකක් තමයි Virtual Box Software එකක් අරගෙන Windows වල Install කරලා Virtual Box එක උඩ Ubuntu හරි නැත්නම් තමන් කැමති මෙහෙයම් පද්ධතියක් අරගෙන වැඩ කරන එක. මේ ක්‍රමයේදි අනිවාර්යයෙන්ම Ubuntu OS එක ඇතුලට යන්න ඕන වුනොත් Windows වලට ගිහින් Virtual Box එක On කරලා අපේ අනිත් මෙහෙයුම් පද්ධතිය Load කරගන්න වෙන්නේ.
හැබැයි මේ ලිපියෙදි මම කියල දෙන්න බලාපොරොත්තු වෙන්නේ වෙන වෙනම මේ මෙහෙයුම් පද්ධති දෙක පරිගණකයේ ස්ථාපනය කරගන්නේ කොහොමද කියලා. මේ ක්‍රමයේදි පරිගණකය Boot වෙනකොට අහනවා ඔයාට මොන මෙහෙයුම් පද්ධතියකින්ද දැන් පරිගණකය Boot කරගන්න ඕන කියලා. ඔයාලට ඒ වෙලාවට අවශ්‍ය කරන මෙහෙයුම් පද්ධතියක් ඒ වෙලාවෙදි තෝර ගන්න පුළුවන්. උදාගරණයක් විදියට පහල තියෙන Image එක බලන්න.

1වෙනි පියවර

මුලින්ම අපිට Ubuntu Install කරගන්න නම් Ubuntu OS එක ඕන වෙනවා. දැනටමත් මොකක් හරි Version එකක් Download කරලා තියෙනවා නම් කමක් නෑ. නැත්නම් Ubuntu Official Site එකට මෙතනින් ගිහින් Download කරගන්න තමන් කැමති Version එකක්. මේ Post එක ලියන දවස වෙනකොට නම් ඇවිත් තියෙන Latest Version එක Ubuntu 17.10.01 හැබැයි මම Recommend කරන version එක 16.04 LTS Version එක. LTS Version එකක තියෙන විශේෂත්වය තමයි මේ වර්ගයේ Ubuntu Version වලට උපරිම අවුරුදු 4ක් විතර වෙනකම් Updates හම්බෙනවා. අනිත් version වලට ගොඩක් වෙලාවට අවුරුද්දක් විතර යනකම් විතරයි Support කරන්නේ.

හරි කැමති Version එකක් Download කරගත්තට පස්සේ කරන්න තියෙන්නේ මේ OS එක Install කරන්න පුළුවන් වෙ විදියේ Bootable Device එකක් හදාගන්න එක. Bootable Device එකක් විදියට අපිට USB Drive එකක් එහෙමත් නැත්නම් DVD එකක් භාවිතා කරන්න පුළුවන්. හැබැයි මම මේ Tute එකේදි භාවිතා කරන්නේ USB Drive (Pen Drive) එකක්. Pen එකක් Bootable කරගන්න අපිට තවත් පොඩි Software එකක් ඕන වෙනවා. ඒ තමයි Rufus. ඒක ඔයාල ළඟ නැත්නම් මෙතනින් Download කරගන්න.

දැන් Rufus On කරගන්න. දැන් ඔයාලට  මේ වගේ Interface එකක් දකින්න පුළුවන් වෙයි.

හරි, මම දැන් පැහැදිලි කරගෙන යන්න හදන්නේ මේ Interface එකේ තියෙන එක එක Options වලින් මොනාද වෙන්නේ කියලා.

Device : මුලින්ම තියෙන Drop Down එක තමයි මේක. මෙතනින් ඔයාට Bootable කරගන්න ඕන Pen Drive එක  Select කරගන්න ඕන.

Partition Scheme and Target System Type: මෙතනින් Select කරන්න ඕන Install කරන්න ඕන Scheme එක. මේකෙදි GPT format එක Select කරනවා නම් හොදයි. මොකද GPT එක තමයි දැනට තියෙන අලුත්ම Scheme එක. හැබැයි සමහර පරණ පරිගණක මේකට Support කරන්නේ නෑ. මොකද GPT Scheme එක අඳුන ගන්න පුළුවන් EFI/UEFT වර්ගයේ Firmware එකක් තියෙනවා නම් විතරයි. MBR එක දැනට අවුරුදු 30ක් විතර භාවිතා BIOS වර්ගයේ Firmware වලට හදපු Scheme එකක්. GPT දාල ඉස්සරහට යන්න බැරි වුණොත් මෙතනින් MBR Select කරන්න.

File System: මේකෙදි තෝර ගන්න Options 2ක් තියෙනවා. FAT32 / NTFS. ඔයා NTFS file type එකෙන් pen එක හැදුවොත් සමහර වෙලාවට මේක UEFI එකට Read කරගන්න බැරි වෙනවා. NTFS එකයි FAT32 එකයි අතර වෙස තමයි FAT 32 වලදි 4GB වලට වැඩි file එකක් තියාගන්න බෑ. හැබැයි අපේ OS එක ඒච්චර ලොකු නැති නිසා අපිට පහසුවෙන් FAT32 එක භාවිතා කරන්න පුළුවන්.

Cluster Size: මේක PEN Dirve එකේ data write කරන විදිය Change කරන්න තියෙන Option එකක්. ඒක වෙනස් කරන්නේ නැතුව ඒ විදියටම තියන්න.

Volume Lable: මේකට කැමති නමක් භාවිතා කරන්න පුළුවන්. ඒ කියන්නේ Pen Drive එක අඳුන ගන්න පුළුවන් නමක් දාගන්න.

අපි තාම Download කරගත්ත Ubuntu ISO file එක Select කරේ නෑ. ඒක කරන්නේ පහලින් තියෙන පොඩි DVD ROM එකේ ICON එක උඩ click කරලා. පැහැදිලි නැත්නම් පහල Image එක බලන්න.

මේ පියවරේදි අපි කරන්න යන්නේ පරිගණකයේ අලුත් OS එක දාන්න හිස් ඉඩක් හදා ගන්න එක. ඇත්තටම හිස් Partition එකක් හදා ගන්න එක. දැන් එක පාරටම ඔළුවට එන්නේ මේක හදා ගන්නේ කොහොමද කියන එකයි කොච්චර Size එකක් ඕනද කියන එකයි. පළවෙනි ප්‍රශ්නෙට උත්තරය පහලට යනකොට හොයා ගන්න පුළුවන් වෙයි. හැබැයි දෙවෙනි ප්‍රශ්නෙට උත්තරේ එක එක්කෙනා අනුව වෙනස් වෙනවා. උදාහරණයක් විදියට මම නිකන් Ubuntu කියන්නේ මොකද්ද කියලා දාලා බලන්න හදන කෙනෙක් නම් ඔයාට OS එකට යන ඉඩට අමතරය තව 5GB විතර වැඩිපුර තිබුණාම ඇති. හැබැයි ඔයා Ubuntu වල දිගටම වැඩ කරන්න හිතන් ඉන්න කෙනෙක් නම් සාමාන්‍යයෙන් අමතර 25GB විතර තියා ගත්තොත් හොඳයි.

හරි දැන් අපි කරන්න හදන්නේ දැනට ඔයාගෙ පරිගණකයේ තියෙන Partition වලට වත් ඒවා ඇතුලේ තියෙන Data වලට වත් හානියක් කරන්නේ නැතුව තවත් පොඩි partition එකකට Data Shrink කරගන්න. ආපහු මේකටත් පොඩි Gadget කෑල්ලක් ඕන වෙනවා. Minitool Partition Wizard. මේක free දෙන Software එකක්. නැත්නම් Download කරගන්න.

Install කරගෙන Open කරගන්න.

මේ Software එක භාවිතා කරලා අවශ්‍ය Partition එක Resize කරගන්න පුළුවන්. ඉතිරි වෙන කොටසින් Unpartition Space එකක් හැදෙනවා. ඒකෙන් අවශ්‍ය Size එක අරගෙන අලුත් partition එකක් Create කරගන්න. අන්තිමට ඔක්කොම කරගෙන ඉවර වුණාම උඩම Menu Bar එකේ තියෙන Apply Button එක Click කරන්න. දැන් Changes ඔක්කොම Hard එකට Write වෙනවා.

3වෙනි පියවර

දැනට අපි ළඟ තියෙනවා Ubuntu Bootable විදියට හදපු Pen Drive එකක්. ඒ වගේම ඒක Install කරගන්න Free Space එකක්. දැන් මේ පියවරේදි කරන්නේ අර pen එකේ තියෙන ෆයිල් ටික අලුත් Partition එකට Install කරන එක. මේ වැඩේ කරන්න Windows වලදි බැහැ. දැන් පරිගණකය PEN එකෙන් Boot කරගන්න ඕන. ඒකට කරන්න තියෙන ලේසිම විදිය තමයි Start Menu එකේ Restart Button එක Shift එක Press කරගෙන Click කරන එක. දැන් මේ විදියේ Insterface එකක් open වෙනවා.

Use a Device Click කරලා Pen එක එතනින් Select කරගන්න. දැන් restart වෙලා Pen එකෙන් computer එක boot වෙනවා.

දැන් මේ වගේ Boot option window එකක් open වෙනවා.

දැන් මෙතනින් Install Ubuntu Option එක select කරගන්න.

මේ විදියට Tick දාලා Next කරන්න. මේ 3rd party apps ඕන වෙනවා Players වලට. Internet Connection එකක් තියෙනවා නම් විතරක් Download Updates එකට tick එකක් දාන්න.

මෙතනින් Something Else click කරලා continue කරන්න.

දැන් මෙතනින් අපි කලින් හදපු Partition එක Select කරලා “–” Button එක Click කරන්න.
