Reading Time: 2 minutes

හැමෝටම ආයුබෝවන් යාලුවනේ මම අද ඔයාලට කියල දෙන්න යන්නේ java කියන programming language එක ගැන.java කියන්නේ object oriented language එකක්.

අද අපි බලමු java install කරගන්නේ කොහොමද කියල.
මුලින්ම ඔය java development kit(jdk) එක download කරගන්න ඕනි.ඒ සදහා link එක මෙතනින් ලින්ක් එකටගියාම ඔයාලට පහල තියෙන විදියෙ page එකක් බලාගන්න පුළුවන්.
ඔයාට පුළුවන් මෙතනින් කැමති java platform එකක් තෝරාගන්න. example එකක් විදියට මම මෙතනින් java platform (jdk)10 තෝරගත්ත කියමුකෝ.එතකොට ඔයාට පහත පරිදි window එකක් දකින්න පුළුවන්.

මෙතනින් ඔයාගේ operating system එකට ගැලපෙන install file එක තෝරගෙන download කරගන්න.ඊට පස්සෙ install කරගන්න.

Install කරල ඉවර උනාම තව දෙයක් තියෙනව කරන්න. ඒ තමයි java path එක හදන එක.
දැන් ඔයා command prompt එකට ගිහින් javac ලෙස type කර enter කරල බලන්න.එතකොට පහත පරිදි window එකක් එන්නෙ නැතුව error එකක් ආවොත්, ඔයාට path එක හදන්න සිදුවෙනවා.

path එක හදාගන්න මේවිදියට යන්න.

my computer >properties >Advanced system setting >Enviroment variables

එවිට ලැබෙන window එකේ පහත highlight කර ඇති new button එක click කරන්න,

ඉන්පසු ලැබෙන window එකේ variable name එකට Path ලෙසද variable value එකට ඔයා java install කරපු Drive එකේ සිට java jdk bin folder එක දක්වාම තිබෙන address එක ලබාදී ok click කරන්න. ඔයා install කරද්දි default settings වෙනස් කරේ නැත්ත්ම්, මේ වගේ path එකක් තමයි තියෙන්න ඕනෙ.

>c:/programe files/java/
